# Sales Trend Analysis Using SQL Aggregations

## 📌 Objective
The objective of this task is to analyze sales trends by calculating monthly revenue and order volume using SQL aggregate functions. This project demonstrates the use of SQL for summarizing and analyzing business data through grouping, aggregation, and sorting techniques.

---

## 🛠️ Tools Used
- MySQL Workbench
- SQL
- GitHub

---

## 📂 Dataset
The dataset consists of an online sales table containing transaction details such as:
- Transaction ID
- Date
- Product Category
- Product Name
- Units Sold
- Unit Price
- Total Revenue
- Region
- Payment Method

## 📚 SQL Concepts Used
- SELECT
- GROUP BY
- ORDER BY
- SUM()
- COUNT()
- COUNT(DISTINCT)
- MONTH()
- Aggregate Functions

---

## 📈 Analysis Performed
The following analyses were performed using SQL:

- Calculated monthly sales revenue
- Counted monthly order volume
- Grouped sales data by month and year
- Sorted the results chronologically
- Identified sales trends over time
